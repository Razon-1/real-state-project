# 🏠 real-state-project

**Note: This is a sample static design without backend. This is just UI for demonstration purposes.**

Modern, responsive real estate website UI built with React.js and Tailwind CSS.

---

## ⚡ Quick Start

```bash
cd frontend/client
npm install
npm start
```

Open: **http://localhost:3000**

---

## 📄 Pages

- **/** - Homepage (Banner, About, Projects, Testimonials, Contact)
- **/blog** - Blog listing page

---

## 🎨 Features

- ✅ 11 React Components
- ✅ Fully Responsive (Mobile, Tablet, Desktop)
- ✅ Tailwind CSS Styling
- ✅ Smooth Animations
- ✅ Interactive Forms (Frontend only)
- ✅ Image Sliders
- ✅ Video Integration
- ✅ WhatsApp Button

---

## ⚠️ Important

- **No Backend** - This is frontend UI only
- **No Database** - All data is static in components
- **Forms** - Show success messages only (no actual submission)
- **Purpose** - UI demonstration and design showcase

---

## 🎨 Customize

### Change Colors
Edit `frontend/client/tailwind.config.js`:
```js
colors: {
  primary: '#003a71',
  secondary: '#ff0000',
}
```

### Update Content
Edit component files in `frontend/client/src/components/`

npm run build
